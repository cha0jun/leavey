# User Guides

This document provides role-based instructions for using the Leavey system.

## 1. Employee Guide (Contractors)

As an employee, you can submit leave requests and track their status.

### Submitting a Leave Request
1.  Navigate to the **New Leave** page.
2.  Select the **Leave Category** (e.g., Annual, Sick).
3.  Choose the **Start Date** and **End Date**.
4.  Provide a **Reason** for the request.
5.  Upload any supporting **Documents** (e.g., medical certificates).
6.  Click **Submit**.

### Viewing Your Leaves
- Use the **My Leaves** dashboard to view the status (Pending, Approved, Rejected) of your submissions.

---

## 2. Manager Guide

Managers are responsible for reviewing and approving leave requests from their team.

### Approving/Rejecting Requests
1.  Go to the **Management** section.
2.  Review the list of **Pending Requests**.
3.  Click on a request to view details and attachments.
4.  Select **Approve** or **Reject**. All actions are logged for audit purposes.

---

## 3. Admin & Finance Guide

Admins and Finance Officers manage the system and generate financial reports.

### User Management
- View and edit user roles (Contractor, Manager, Admin).
- Update user metadata (Vendor ID, Department).

### Monthly Reconciliation
1.  Navigate to the **Finance** dashboard.
2.  Select the **Year** and **Month**.
3.  Enter the **Working Days** for that month.
4.  Review the calculated **Billable Days** for each contractor.
5.  Click **Export CSV** to download the report for invoicing.

> [!NOTE]
> The Finance dashboard displays real-time calculations based on the snapshot of the leave category's chargeable status at the time of submission.
